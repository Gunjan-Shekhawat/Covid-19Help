# Covid-19Help

A Website containing all the information regarding Covid-19.
[Check the website.] (https://covid-19help.herokuapp.com)

## Tech Stack

- NodeJS
- Express
- MongoDB

Deployed on Heroku.

## How to run

1. Clone Repo
   - `$> git clone https://github.com/shivamgupta1999/Covid-19Help.git`
2. Open Terminal
3. Enter command: node app.js
